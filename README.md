## 仅供预览和参考
本站经过定制, 加入了某些你可能不需要的东西(比如说`yuque-hexo`, `algolia 搜索`, `GitHub Actions CI`等等), 这些东西需要经过比较复杂的配置, 不能直接拿来使用, 所以你可以预览和参考, 但是如果想自己搭建博客, 请你最好还是安装[Volantis X官方的最新版主题](https://github.com/volantis-x/hexo-theme-volantis)
